Get Started

Get Started

To install:

npm install --save native-base-web

NativeBase-web is depended on react-native-web-extended, an extended version of react-native-web. Install it if you have not yet configured your app structure:

npm install --save react-native-web-extended

Setup resolve alias in your webpack configuration:

resolve: {
    alias: {
        'react-native': 'react-native-web-extended',
        'native-base' : 'native-base-web'
    }
}
